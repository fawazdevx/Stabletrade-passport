Public app

The Node server serves `frontend/dist` by default. Legacy `public/` only runs with `SERVE_LEGACY_PUBLIC=true`.

Unsafe endpoints

Demo mutations, fallback receipts, and uploads return `403` unless an explicit local flag or admin bearer token is present.

Settlement control

The V3 proxy upgrade must be deployed before the final recording so delivery proof is enforced onchain.
